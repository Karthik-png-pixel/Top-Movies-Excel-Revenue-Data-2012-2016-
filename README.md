# Project: Movie Revenue Analysis Using Excel

## Objective: Analyzed movie industry data across a timeframe of 5 years to identify box office performance, profitability, and key factors such as directors, actors and genres.

## Key Analysis Performed: Revenue trends by year, Performance of low-budget films (<$10M), Average movie budget by director and cast member, and Genre-based revenue analysis

## Tools Used: Google Sheets, Pivot Tables, Data Visualization (Bar Charts)
